# Directory contents

This directory contains saved jupyterLab workspace layouts, the **Dockerfile** and **docker-compose** file as well as the **entrypoint** file. 

The files **pycram-http.rosinstall** and **requirements.txt** are needed in the Dockerfile.
They are included here locally as they were removed from the original git repo.

The file **jupyter-config.json** is also needed in the Dockerfile.
